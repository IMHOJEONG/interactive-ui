- 브라우저에서 기본적으로 정의되어 있는 `사용자 에이전트 스타일시트(User Agent Stylesheet)`에 의해 각각의 스타일이 적용됨 

    - 각 브라우저 개발사는 자체적인 디자인 가이드라인에 따라 기본 스타일을 다르게 적용

### 의미론적 태그(Semantic Tag)

- HTML에서 콘텐츠의 의미를 명시적으로 정의하는 요소 



### a 태그 

- `target` 속성: 링크가 열리는 방식을 제어 

    - `_blank`: 새 탭 / `_self`: 현재 탭 / `_parent`: 부모 프레임에서 / `_top`: 최상위 프레임에서 링크를 열게 함 

- `rel` 속성: 링크의 관계와 보안 설정을 정의 

    - 'nofollow': 검색 엔진 크롤러의 링크 추적을 방지 

    - 'noopener', 'noreferrer': 새 탭에서 열린 페이지가 원래 페이지에 접근하는 것을 차단하여 보안을 강화 

    - 'external': 외부 링크임을 명시 

### img 태그 

- alt 속성: 웹 접근성, 접근성 측면에서 매우 중요 / 이미지를 볼 수 없는 상황에서 이미지를 설명하는 대체 텍스트를 제공 


### form 태그 

- 사용자 입력을 수집하고, 서버로 전송하기 위한 컨테이너 역할을 함 

- 이 태그 내부에는 다양한 입력 요소들이 포함될 수 있음 & 사용자로부터 다양한 형태의 데이터를 수집
